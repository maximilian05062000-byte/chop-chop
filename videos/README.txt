Инструкция по загрузке видео для Телевизора Чоп-Чопы:

1. Загрузите сюда ваши смешные или познавательные видеоролики в формате MP4 (.mp4).
2. Называйте файлы согласно дате, когда это видео должно показываться:
   - По точной дате: ГГГГ-ММ-ДД.mp4 (например, 2026-07-12.mp4)
   - По дню и месяцу (будет повторяться каждый год): ДД-ММ.mp4 (например, 12-07.mp4)
3. Добавьте файл default.mp4 — это видео будет показываться по умолчанию в любой день, для которого вы не загрузили отдельный ролик.

При обновлении сайта на GitHub Pages убедитесь, что вы загружаете папку "videos" целиком вместе с файлом index.html!
